# ML Systems

## Overview

Front door for modern ML engineering. This skill groups neural training systems, transformer workflows, graph models, reinforcement learning stacks, and optimization-heavy ML workflows.

## Use This For

- PyTorch Lightning training loops
- Hugging Face Transformers workflows
- graph neural networks
- reinforcement learning experiments
- multi-objective optimization tied to ML systems
- Python execution and testing in ML contexts

## Do Not Use This For

- tabular empirical analysis or econometrics: use `empirical-analysis-python`
- time-series forecasting as the main objective: use `time-series-analysis`
- figures and communication outputs: use `visualization` or `research-graphics`

## Internal Routing

- deep learning orchestration
- transformer inference and fine-tuning
- graph ML
- RL stacks
- optimization support

## Operating Rule

If the problem is about training or operating a model system rather than interpreting a finished empirical result, start here.

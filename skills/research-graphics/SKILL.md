# Research Graphics

## Overview

Front door for visual assets that are not ordinary data charts. This skill handles diagrams, conceptual figures, infographics, and AI-generated illustrations used in papers, talks, and reports.

## Use This For

- model schematics
- workflow diagrams
- conceptual illustrations
- infographics
- polished non-chart figures for papers or talks

## Do Not Use This For

- data-driven plots: use `visualization`
- full slide decks or posters as the main deliverable: use `slides-and-teaching`
- document conversion: use `documents-and-conversion`

## Internal Routing

- scientific schematics
- general image generation and editing
- infographic design

## Operating Rule

If the output is a visual asset rather than an analytical chart, start here.

# Research Writing

## Overview

Front door for long-form research prose. This skill covers papers, proposals, reports, venue-specific packaging, and economics-style or AI-research-style manuscript development.

## Use This For

- drafting a paper
- restructuring a manuscript
- writing a grant proposal
- preparing a submission package
- writing a research report or market-style memo
- turning research into polished prose

## Do Not Use This For

- finding sources: use `research-discovery`
- reviewing or grading finished work: use `research-review`
- code implementation plans: use `superpowers:writing-plans`
- slide decks, lectures, or posters as the main deliverable: use `slides-and-teaching`

## Internal Routing

- manuscript writing
- grant writing
- venue constraints and templates
- economics and finance paper structure
- report-style outputs

## Operating Rule

If the core task is “write the research deliverable,” start here.

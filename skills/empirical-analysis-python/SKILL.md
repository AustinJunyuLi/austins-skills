# Empirical Analysis Python

## Overview

Front door for Python-side empirical work. This skill covers exploratory analysis, classical and Bayesian modeling, inference, diagnostics, explainability, dimensionality reduction, and exact symbolic helpers where they support empirical research.

## Use This For

- exploratory data analysis
- regression and inference
- model diagnostics and robustness checks
- Bayesian estimation with PyMC
- tabular machine learning for empirical work
- SHAP-based interpretability
- UMAP embeddings for exploratory structure
- symbolic derivation helpers tied to the analysis

## Do Not Use This For

- forecasting or sequence-first work: use `time-series-analysis`
- deep learning, transformers, RL, or graph training: use `ml-systems`
- file conversion or spreadsheet surgery: use `documents-and-conversion`
- R-first workflows: use `data-analysis-r`

## Internal Routing

- EDA
- statsmodels-style inference
- PyMC and Bayesian workflows
- scikit-learn-style tabular modeling
- explainability and embeddings

## Operating Rule

If the primary question is “analyze this dataset in Python and defend the result,” start here.

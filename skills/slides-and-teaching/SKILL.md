# Slides And Teaching

## Overview

Front door for talks, lectures, and teaching artifacts. This skill covers deck creation, lecture construction, Beamer and Quarto workflows, posters, lecture notes, and paper-to-presentation transformations.

## Use This For

- research talks
- lecture decks
- seminar slides
- course notes derived from slides or papers
- Beamer to Quarto translation
- slide deployment workflows
- poster creation when the main goal is presentation output

## Do Not Use This For

- auditing an existing deck: use `slide-review`
- pure file conversion without presentation design intent: use `documents-and-conversion`
- manuscript drafting: use `research-writing`

## Internal Routing

- slide creation
- lecture authoring
- lecture-note generation
- Beamer and Quarto tooling
- posters and presentation derivatives

## Operating Rule

If the output is for a talk, class, seminar, or poster session, start here.

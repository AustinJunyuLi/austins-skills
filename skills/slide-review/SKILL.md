# Slide Review

## Overview

Front door for reviewing presentation materials. This skill covers proofreading, pedagogical review, visual audit, adversarial critique, and Quarto-vs-Beamer QA.

## Use This For

- visual QA on slides
- pedagogy checks
- proofreading decks
- overflow and consistency audits
- adversarial challenge of a talk
- parity checks between presentation formats

## Do Not Use This For

- creating the deck: use `slides-and-teaching`
- reviewing manuscripts or grants: use `research-review`

## Internal Routing

- proofreading
- pedagogy review
- visual audit
- devil’s-advocate critique
- cross-format QA

## Operating Rule

If the user already has slides and wants them evaluated or pressure-tested, start here.

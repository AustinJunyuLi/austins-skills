# Data Analysis R

## Overview

Front door for R workflows. This skill covers end-to-end empirical analysis in R, including script execution, review of R analysis code, and production of tables and figures.

## Use This For

- R-first research pipelines
- running and checking R scripts
- regression and table production in R
- data wrangling and figure generation in R
- reviewing or tightening R analysis code

## Do Not Use This For

- Python-first analysis: use `empirical-analysis-python`
- time-series-specific modeling: use `time-series-analysis`
- general document conversion: use `documents-and-conversion`

## Operating Rule

If the intended implementation language is R, or the user is already working from an R script or R project, start here.
